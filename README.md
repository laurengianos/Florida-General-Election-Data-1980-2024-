# Florida General Election Data 1980-2024

Voter turnout has increased with the growing polarity plaguing U.S. politics. Now more than ever, voter education needs to be a main priority -- regardless of who you vote for. Many see Florida as a "deep red" state, but Florida has not always been this way, and an argument can be made that it's not "deep red" at all, it just appears that way. But once you dig deeper, in which I did  here, it's irrefutable that Florida has a large Democratic base that might need motivation to be mobilized.
These visualizations are just a stepping stone for further research down the line.
